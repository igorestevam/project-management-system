# Projeto de API REST com Spring Boot e JPA

Este projeto foi desenvolvido como parte do curso "Java Backend 360°: Web Services API com Spring Boot e JPA", ministrado por Carlos Tosin na Udemy. O objetivo do projeto é criar uma API RESTful utilizando o framework Spring Boot com persistência de dados via JPA.

## Tecnologias Utilizadas

- **Spring Boot**: Framework para construção de aplicações Java de forma rápida e produtiva.
- **JPA (Java Persistence API)**: Especificação para acesso a dados e manipulação em linguagem Java.
- **MySQL**: Sistema de gerenciamento de banco de dados utilizado para armazenar os dados da aplicação.
- **Postman**: Ferramenta para testes de APIs.

## Funcionalidades

- Criação de endpoints para gerenciar recursos (CRUD).
- Integração com banco de dados MySQL.
- Uso de anotações do Spring para gerenciar a lógica de negócio e os endpoints REST.

## Instruções de Instalação

1. Clone este repositório:
    ```bash
   git clone https://github.com/igorestevam/project-management-system.git
2. Navegue até a pasta do projeto:
   ```bash
   cd project-management-system
3. Configure seu banco de dados MySQL e atualize as credenciais no arquivo application.yml.
  
4. Execute a aplicação:
   ```bash
   mvn spring-boot:run
   
## Testando a API

Use o Postman para testar os endpoints da API. Certifique-se de configurar as requisições corretamente para interagir com os dados.

## Conclusão

Este projeto é um exercício prático das habilidades adquiridas no curso e demonstra a criação e gestão de uma API utilizando boas práticas de desenvolvimento em Java com Spring Boot.
